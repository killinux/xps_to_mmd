# 胸部物理调研报告

> 调研日期: 2026-04-26
> 来源: RGBA 博客 (权威)、もにゃ 实测、Yahoo 知恵袋、PMXEditor 社区
> 结论: **当前代码有两个错误需要修正**

---

## 一、四种胸部物理架构

| 架构 | 复杂度 | 刚体数/侧 | 稳定性 | 效果 | 适合自动生成 |
|------|--------|----------|--------|------|------------|
| **A: 单体** | 低 | 1 | 最高 | 标准弹跳 | **是（推荐）** |
| B: RGBA式 | 高 | 5 | 低 | "ボヨヨーン"果冻感 | 否（需手调） |
| C: AH式 | 高 | 多体 | 中 | "プリン"布丁感 | 否（需加骨） |
| D: 双体链 | 中 | 2 | 中 | 链式摆动 | 可选 |

**结论：架构 A（单体）是自动生成的正确选择。** 90% 的 PMX 模型使用这种架构。

### RGBA式（架构 B）简介

每侧 5 个刚体：主体 + 后方辅助 + 前方辅助 + 旋转辅助 + 前后辅助。通过多关节组合平移和旋转，实现果冻感。**仅在 MikuMikuDance 中生效**（不支持 MikuMikuMoving），且需要手工调参。不适合自动化。

## 二、Dynamic vs Dynamic+Bone（关键发现）

### 结论：**胸部必须用 Dynamic (type=1)，不能用 Dynamic+Bone (type=2)**

RGBA 原文："移動で動いているおっぱい剛体等には使えません"

| | Dynamic (type=1) | Dynamic+Bone (type=2) |
|---|---|---|
| 传递旋转 | ✓ | ✓ |
| 传递平移 | **✓** | **✗（每帧强制重置位置）** |
| 胸部效果 | 自然弹跳 | 僵硬，只能转不能移 |
| 适用场景 | **胸部、裙子末端** | 前刘海防抖、配饰锚点 |

**当前代码 `MODE_DYNAMIC_BONE` (type=2) 是错误的。** 胸部物理的核心是平移（重力下坠 + 惯性晃动），Dynamic+Bone 会丢弃平移只保留旋转。

## 三、推荐参数（社区共识）

### 刚体参数

| 参数 | 当前值 | **推荐值** | 来源 |
|------|--------|----------|------|
| Type | Dynamic+Bone (2) | **Dynamic (1)** | RGBA 明确指出 |
| Shape | Sphere | Sphere | 共识 |
| Mass | 0.5 | **1.0** | もにゃ实测 + 多来源 |
| Linear Damping | 0.5 | 0.5 | OK |
| Angular Damping | 0.5 | 0.5 | OK |
| Friction | 0.0 | 0.0 | OK |
| Bounce | 0.0 | **0.0（绝不超过 0.05）** | Amenrenet 警告 |

### 关节参数

| 参数 | 当前值 | **推荐值** | 来源 |
|------|--------|----------|------|
| Rotation X (前后) | ±5° | **±10°** | 社区中位值 |
| Rotation Y (扭转) | ±5° | **±3°** | 扭转应最小 |
| Rotation Z (左右) | ±5° | **±5°** | OK |
| Spring Angular | **2000** | **100** | もにゃ: 2000 太硬,100=硬弹力 |
| Spring Linear | 0 | 0 | OK |
| Translation | 0,0,0 | 0,0,0 | OK |

### Spring 值的影响（もにゃ实测 4 种预设）

| 预设 | Mass | Damping | Spring | 效果 |
|------|------|---------|--------|------|
| 重+软 | 0.1 | 0.99 | 0 | 缓慢飘浮，梦幻感 |
| 重+弹 | 0.1 | 0.99 | 100 | 缓慢弹回，沙包感 |
| 轻+快 | 1.0 | 0.1 | 0 | 快速响应，自由 |
| **轻+弹** | **1.0** | **0.1** | **100** | **快速弹回，有力** |

当前 spring=2000 会导致胸部几乎不动（弹簧力远大于重力）。降到 100 后有明显弹力但不会失控。

## 四、已知陷阱

| 陷阱 | 原因 | 解决 |
|------|------|------|
| 中线塌陷 | 两侧刚体向中心滑动 | 上半身2 碰撞体足够大作为墙 |
| 穿透身体 | 碰撞体太小 | 身体碰撞体略大于 mesh |
| 静止抖动 | mass<0.1 或 spring 过高 | mass≥0.5, spring≤200 |
| 僵硬不动 | 用了 Dynamic+Bone | 改 Dynamic |
| 重力下垂 | mass 高无回弹 | spring=100 提供回弹力 |

## 五、需要修改的代码

```python
# physics_operator.py

# 1. 刚体类型: Dynamic+Bone → Dynamic
dynamics_type=MODE_DYNAMIC  # 不是 MODE_DYNAMIC_BONE

# 2. 弹簧常数: 2000 → 100  
BREAST_SPRING_ANGULAR = 100.0  # 不是 2000

# 3. 关节旋转限制: 需要按轴分别设置
#    X(前后): ±10°, Y(扭转): ±3°, Z(左右): ±5°

# 4. 质量: 0.5 → 1.0 (可选，更自然)
mass=1.0
```
